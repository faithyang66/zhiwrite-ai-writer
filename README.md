# 智写AI写作助手（ZhiWrite AI Writer）

在 WordPress 后台使用 AI 生成/改写文章，并提供“可控、可观测、可扩展”的工作流：支持多接口/多模型切换、生成队列与并发控制、日志与导入导出、SEO 字段与结构化内容增强、分类/标签建议与可选写入确认、Webhook 回调与重试队列、缓存与成本观测等。

## 适用场景
- 批量或半自动生成文章草稿
- 需要统一写作风格与结构（模板/风格预设）
- 需要记录生成过程并可回溯（日志/诊断导出）
- 需要与外部系统同步（Webhook 回调 + 签名 + 重试）

## 功能亮点
### 生成体验
- 支持多接口配置（API base_url / API Key / 默认模型）
- 支持同步生成或后台队列（WP-Cron）模式
- 结果区支持“仅展示结果”下可编辑后保存/发布，并保留原始生成版本用于回滚
- 支持生成页面的模板/风格/变量“提示词构建器”（Prompt Builder）

### 内容增强（生成后解析与写入）
- SEO 字段生成（slug、焦点关键词等）
- 自动目录（TOC）
- 自动 FAQ（问答结构化块）
- 自动摘要/导语（可选）
- 内部链接策略（基于站内公开文章生成推荐并插入正文）
- 分类/标签建议（支持人工确认、可选自动写入文章）
- “生成记录/日志”导出（JSON/CSV）与诊断导出（脱敏）

### 可观测与安全
- 生成日志记录（包含 usage/token（如上游提供））
- 统一错误码与管理员可读的失败信息
- 审计与诊断导出（脱敏；包含最近错误与环境信息）
- 敏感配置保护（API Key 不回显；空输入保留旧值等）
- PII 脱敏与外发字段禁用（可配置）
- 内容安全检测（本地规则 + 可选外部检测，含降级策略）

### 成本控制与缓存
- 缓存策略（transient）避免重复扣费
- “用量统计”页面：统计请求数/耗时/tokens（基于带 tokens 的成功调用）
- 缓存命中统计（估算节省）
- 缓存清理工具：一键清理本插件缓存

### Webhook
- 生成事件回调（成功/预览/失败等）
- HMAC-SHA256 签名（可配置启用）
- Webhook 失败进入重试队列（指数退避 + 最大次数 + 死信保留）
- Webhook 队列与日志面板：可手动触发重试、查看回调记录（耗时/HTTP 状态/脱敏片段/签名启用情况）

### 文章列表增强
- `edit.php` 文章列表：一键生成并保存标签（可选覆盖/增量写入）
- 支持批量生成标签（队列执行 + 进度轮询 + 列表刷新为服务端渲染 HTML）

## 系统要求
- WordPress：至少 6.0
- PHP：至少 7.4

## 安装
1. 将本插件解压到 `wp-content/plugins/zhiwrite-ai-writer/`
2. WordPress 后台启用插件：`智写AI写作助手`

## 配置与使用（建议顺序）
1. 进入 `智写AI -> 功能配置`：
   - 配置接口（API 地址 / Key / 默认模型）
   - 配置生成策略与安全策略（PII、内容安全、外发字段禁用等）
   - 配置 SEO/TOC/FAQ/内部链接/分类标签写入策略
2. 进入 `智写AI -> 生成内容`：
   - 选择接口（可选覆盖模型）
   - 填写写作需求（prompt）
   - （可选）选择模板/风格/变量覆盖
   - 点击“开始生成”
3. 在生成完成后：
   - 如果策略是“仅展示结果”，可在右侧编辑后保存/发布
   - 可在“分类/标签建议”区块勾选后应用到文章（按设置写入）
4. 需要排障时使用：
   - `智写AI -> 生成记录`
   - `智写AI -> 用量统计`
   - `智写AI -> Webhook 日志 / 队列`

## 提示词构建器（Prompt Builder）
功能配置页新增 Tab：**提示词构建器**。
- 以 JSON 管理：模板 / 风格预设 / 变量
- 保存会保留最近历史版本（可选择回滚）
- 生成页可选择模板、风格，并支持本次变量覆盖（JSON）

示例（变量占位符）：
- `{{site_name}}`
- `{{date}}`
- `{{title}}`
- `{{prompt}}`

## 打包与加密（可选）
本项目提供一个 Windows BAT 脚本用于生成两份 zip：
- 完整插件包：`*_full.zip`
- 加密输入包（仅包含需要加密的 PHP 文件，便于只上传 PHP）：`*_encrypt_php_input.zip`

BAT 文件位置：
- `pack_zhiwrite-ai-writer.bat`（放在站点根目录 `d:\phpstudy_pro\WWW\wpmb.cn\` 下）

生成产物目录：
- `dist/zhiwrite-ai-writer/`

> 说明：加密逻辑由你在“加密后的 PHP 包”中完成；脚本仅负责打包出两份输入。
